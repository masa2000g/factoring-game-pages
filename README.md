# 因数分解マスター

パズルで因数分解を体感する、中学生向けの操作型パズルゲームです。ブラウザだけで動作する静的サイトとして構成されています。

## 概要（OGP用）

> パズルで因数分解を体感しよう！タイルを並べて、ペアを見つけて、共通因数をくくり出す。中学生向けの操作型パズルゲーム。

## 3つのゲームモード

### 🔍 ペアファインダー
- **操作**: 数字バブルを2つタップして選ぶ
- **ルール**: 「かけて c、たして b」になるペアを見つける
- **学べること**: 和と積の関係、因数ペアの感覚
- **間違えたとき**: 「かけ算は合ってる！でもたし算が…」のように具体的に理由を表示

### 🧩 面積パズル
- **操作**: タイル（x², 3x, 6 など）をタップで選んで 2×2 のマスに配置
- **ルール**: 長方形の面積モデルを完成させる
- **学べること**: 面積モデルによる因数分解の視覚的理解
- **正解すると**: マスの辺に因数 (x+2)(x+3) が表示される

### 🎯 共通因数バンドル
- **操作**: 共通因数の候補ボタンを選んで「これでくくる！」
- **ルール**: すべての項を割り切れる最大の共通因数を選ぶ
- **学べること**: 共通因数の発見とくくり出し
- **間違えたとき**: 「3でもくくれるけど、xも共通因数だよ！」のようにヒントを表示

## 特徴

- テキスト入力なし — すべてタップ操作で完結
- **初回チュートリアル** — 初めてのユーザーに遊び方をステップごとに紹介（スキップ可能）
- **モード別ガイド** — 各モード初回プレイ時に操作手順をバナー表示
- **視覚ヒント** — 押せる場所が光って教えてくれる（パルスアニメーション＆タップ手アイコン）
- **やさしい初問** — 各モードの1問目はかんたんな問題で操作に慣れる
- 正解時にコンフェティ＆連続正解ポップアップ＆次のアクション誘導
- 間違えたときは「なぜ違うか」＋「どうすればいいか」を具体的にフィードバック
- スマホ対応（タッチ操作に最適化）
- ポップで楽しいデザイン

## GitHub Pages での公開方法

1. このリポジトリを GitHub にプッシュする
2. リポジトリの **Settings** → **Pages** を開く
3. **Source** で `Deploy from a branch` を選択
4. **Branch** で `main` / `/ (root)` を選択して **Save**
5. 数分後に `https://<username>.github.io/<repo-name>/` で公開される

## ローカルでの確認

```bash
# Python の場合
python3 -m http.server 8000

# Node.js の場合
npx serve .
```

ブラウザで `http://localhost:8000` を開く。
