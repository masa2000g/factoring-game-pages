# 因数分解マスター

ペアファインダーで因数分解を体感する、中学生向けの操作型パズルゲームです。ブラウザだけで動作する静的サイトとして構成されています。

## 概要（OGP用）

> パズルで因数分解を体感しよう！かけ算とたし算が合うペアを見つけて因数分解。中学生向けの操作型パズルゲーム。

## 遊び方

1. 画面に式 x² + bx + c が表示される
2. 「かけて c、たして b」になる2つの数を見つける
3. 数字ボタンを2つタップして選ぶ
4. 正解すると (x+p)(x+q) が表示される → 「次の問題」へ

## 特徴

- **1画面完結** — スクロール不要、問題・選択・結果・次へボタンがすべて見える
- **タップだけ** — テキスト入力なし、2つの数字を選ぶだけ
- **同数ペア対応** — x²+6x+9 → 3と3 のように同じ数を2回使う完全平方式にも対応
- **初回チュートリアル** — はじめて遊ぶユーザーにルールを1画面で紹介
- **1問目ガイド** — ステップごとに「何をすればクリアか」を案内
- **わかりやすいフィードバック** — 間違えたとき「かけ算は合ってる！でもたし算が…」のように具体的に理由を表示
- **正解演出** — コンフェティ＆連続正解ポップアップ
- **スマホ最適化** — タッチ操作に最適化したレスポンシブデザイン

## GitHub Pages での公開方法

1. このリポジトリを GitHub にプッシュする
2. リポジトリの **Settings** → **Pages** を開く
3. **Source** で `Deploy from a branch` を選択
4. **Branch** で `main` / `/ (root)` を選択して **Save**
5. 数分後に `https://<username>.github.io/<repo-name>/` で公開される

## ローカルでの確認

```bash
# Python の場合
python3 -m http.server 8000

# Node.js の場合
npx serve .
```

ブラウザで `http://localhost:8000` を開く。
